### Prerequisites

To run this project, you'll need to have the following installed:

- Node.js : [https://nodejs.org](https://nodejs.org)

- npm :
  ```sh
  npm install npm@latest -g
  ```

### Installation & Start

To run this proejct:

```sh
 npm install
```

```sh
 npm start
```

To test:

```sh
 npm test
```
